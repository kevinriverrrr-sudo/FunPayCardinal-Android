# Руководство по внесению вклада

Спасибо за интерес к FunPayCardinal Android! Мы приветствуем вклад от всех.

## Как начать

1. Форкните репозиторий
2. Клонируйте форк: `git clone https://github.com/ваш-username/FunPayCardinal-Android.git`
3. Откройте проект в Android Studio
4. Соберите проект: `./gradlew assembleDebug`

## Процесс внесения изменений

1. Создайте ветку: `git checkout -b feature/ваша-функция`
2. Внесите изменения
3. Протестируйте изменения: `./gradlew test`
4. Закоммитьте: `git commit -m 'Добавлена новая функция'`
5. Отправьте: `git push origin feature/ваша-функция`
6. Создайте Pull Request

## Стандарты кода

- Используйте Kotlin для нового кода
- Следуйте официальным рекомендациям Android
- Используйте Jetpack Compose для UI
- Добавляйте комментарии на русском языке
- Пишите тесты для новой функциональности

## Форматирование коммитов

- `[FEAT]` - Новая функция
- `[FIX]` - Исправление ошибки
- `[UI]` - Изменения интерфейса
- `[DOCS]` - Изменения в документации
- `[REFACTOR]` - Рефакторинг
